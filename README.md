## About The Project

This is a whole IM project includes both the front-end part (Android) and back-end side.

### Built With
**For Client part:**

- Retrofit
- Butterknife
- Glide
- Dbflow
- Aliyun OSS

**For Server part:**

- Hibernate
- MySQL
- Gson
- Tomcat
